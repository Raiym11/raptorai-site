# Как автоматизировать AmoCRM с помощью n8n

AmoCRM — одна из самых популярных CRM в России, и многие компании тратят часы на рутинные задачи: перенос заявок, смену статусов, отправку уведомлений. В этой статье разберём как автоматизировать эти процессы с помощью n8n.

## Что понадобится

- Аккаунт AmoCRM с правами администратора
- n8n (self-hosted или облачный)
- 30-60 минут свободного времени

## Шаг 1: Подключение AmoCRM к n8n

Для интеграции AmoCRM с n8n используем встроенный нод AmoCRM или HTTP Request для работы с API напрямую.

### Через OAuth

1. Зайдите в настройки AmoCRM → Интеграции
2. Создайте новую интеграцию, получите Client ID и Client Secret
3. В n8n создайте Credentials → AmoCRM OAuth2
4. Авторизуйте интеграцию

## Шаг 2: Автоматическое создание сделок

Создадим workflow который автоматически создаёт сделку при получении заявки с сайта через webhook.

```
Webhook → Set fields → AmoCRM (Create Deal) → Telegram notification
```

Когда клиент заполняет форму на сайте — сделка создаётся автоматически, а менеджер получает уведомление в Telegram.

## Шаг 3: ИИ-квалификация лидов

Самая мощная автоматизация — добавить GPT для квалификации входящих заявок.

Workflow выглядит так:

1. Новая заявка попадает в AmoCRM
2. n8n забирает данные сделки через webhook
3. GPT-4o анализирует заявку и определяет приоритет
4. Сделка автоматически получает нужный тег и статус
5. Горячий лид — менеджер получает срочное уведомление

## Результат

После внедрения такой автоматизации типичная компания экономит 2-3 часа в день на рутинных задачах и повышает скорость обработки заявок на 60-80%.
